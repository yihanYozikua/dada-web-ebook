# CSS偽元素 & 偽類

* 偽元素
  * 在原來的html元素中，用css加入其他元素
  * 並不是網頁裡面的真正元素，但行為有表現的跟真正的元素一樣
  * 以CSS來操控
  * 對SEO沒有幫助，因為並不真正存在於html中
<br>

* 偽類
  * 表達元素的動作、樣式
  * 例如：滑鼠滑過時的樣式、超連結被點過之後的樣式

<br>

|       | 英文名稱 | 符號 |  |  |  |
| ----- | --------------- | -- | ----- | --------------- | ----------------------- |
| 偽元素 | Pseudo Elements |  ::  | 是元素 | 會出現在DOM tree | 用來選擇元素的 **指定部分** |
| 偽類 | Pseudo Classes |  :  | 不是元素 | 不會出現在DOM tree | 用來表達元素的 **狀態** |
