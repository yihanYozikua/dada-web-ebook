# html tag

### <head>
```html
<head>
    <!-- 在瀏覽器 tab 標籤上的標題 -->
    <title>網頁標題</title>
    <!-- 聲明該頁面的編碼方式 -->
    <meta charset="UTF-8"> 
    <!-- 行動裝置的顯示方式，initial-scale=1.0 代表不縮放 -->
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <!-- 在瀏覽器 tab 標籤上的小圖示 -->
    <link rel="icon" href="favicon.ico">
    <!-- 連接 main.css -->
    <link href="main.css" rel="stylesheet">
    <!-- 連接 common.js -->
    <script src="common.js"></script>
</head>
```


### <body>
```html
<body>
    <header> 放網頁最上方的資訊，如：Logo / 導航列
        <nav>導航列</nav>
    </header>
    
    <main> 放主要內容
        <article>放主要文章</article>
        <section>分段落，自己定義的段落</section>
        <aside>放一些不是主要資訊的資訊</aside>
    </main>
    
    <footer>放頁面底部的資訊，如：地址 / 聯絡電話 / Copyright
    </footer>
</body>
```



### 列舉
```html
<ul>符號式列舉
    <li>list 1</li>
    <li>list 2</li>
    <li>list 3</li>
</ul>
<ol>數字式列舉
    <li>list 1</li>
    <li>list 2</li>
    <li>list 3</li>
</ol>

<dl>特殊專有名詞說明
    <dt>名稱title</dt>
    <dd>列舉說明</dd>
    <dd>列舉說明</dd>
    <dd>列舉說明</dd>
</dl>
```

### Link
* target
    * ```_self```: 直接在本頁面跳轉
    * ```_blank```: 開新視窗

```html
<a href="網址" target="_self">超連結名稱</a>
<a href="網址" target="_blank">超連結名稱</a>
<a href="#id名稱">跳到某個#id</a>
<!-- 可以利用 <a> 和 #id 跳到某個網頁中同一頁面的特定位置 -->

<button type="button" onclick="你寫的按鈕動作()">
按鈕文字
</button>
<!-- "你寫的按鈕動作()" 這個function要在JS裡面寫 -->
```

### 表單
* ```type```=
    * ```text```：純文字
    * ```email```：會做簡單驗證
    * ```password```：會以替代符號出現
    * ```search```：搜尋框
    * ```tel```：電話
    * ```url```：網址
    * ```radio```：單選
        * ```name```：題目分類
        * ```value```：選項值
        
* 表單屬性（寫在每個```<input>``` ```<textarea>``` ```<select>```裡面）
    * ```name```：```<form>```中每個```<input>```都要有```name```
    * ```value```：預設內容
    * ```placeholder```：提示文字，一但user開始輸入就會消失
    * ```disabled```：禁止填寫
    * ```required```：必填
    * ```maxlength```：輸入框最大字數控制，適用於```text```和```password```
    * ```checked```：預設勾選，適用於```radio```和```checkbox```
    * ```selected```：預設選項，適用於```<option>```

### JS怎麼拿到```<form>```的值
```js
// 以選擇題（radio / checkbox）而言
var ans = document.getElementById("myForm");
// var ans = document.forms[0]; => 上面那行也可以這樣寫
var txt="";
var i;

for( i = 0; i < ans.length; i++ ){
    if( ans[i].checked ){
      txt = txt + ans[i].value + "  ";
    }
  }
console.log( txt );
```
